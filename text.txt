Сложная модульная сетка реализована с помощью продвинутого CSS Grid (адаптивная сетка из 8 карточек, объединение строк).

Использован Flexbox для выравнивания внутреннего контента (шапка, списки, элементы навигации).

Кастомная стилизация скроллбара, нестандартных чекбоксов и градиентов.

Технологии: HTML5, CSS3, CSS Grid, Flexbox.
