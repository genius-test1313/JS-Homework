1. Название работы (выбирай любой вариант):
Вариант 1 (понятный клиенту): Верстка стильного лендинга для салона красоты

Вариант 2 (с фокусом на навыки): Адаптивная промо-страница (HTML/CSS, Absolute Positioning)

Вариант 3 (как на GitHub): Лендинг "The Day Beauty" (Pixel-Perfect верстка)

2. Описание работы (скопируй этот текст):
Выполнила верстку современного и яркого лендинга (промо-страницы) для косметологической клиники и SPA на основе макета из Figma.

Особенности проекта:

Точная (Pixel-perfect) верстка дизайна.

Активное использование абсолютного позиционирования (position: absolute) для сложных декоративных элементов и перекрывающихся блоков.

Верстка нестандартного таймера обратного отсчета (акционный блок).

Построение гибкой структуры страницы с использованием Flexbox и CSS Grid.

Технологии: HTML5, CSS3, Flexbox, CSS Grid.
