Выполнила чистую и современную верстку многостраничного портала для поиска недвижимости на основе макета из Figma. Проект отличается сложной структурой и обилием интерактивных элементов.

Особенности реализации:
✔️ Pixel-perfect перенос дизайна из Figma в код.
✔️ Сложная сетка (CSS Grid) для карточек товаров и каталога.
✔️ Реализация полноценного дашборда пользователя (Личный кабинет) с фиксированным (sticky) боковым меню.
✔️ Кастомная стилизация форм, радиокнопок и ползунков.
✔️ Интерактивные hover-эффекты для кнопок, карточек и социальных сетей.
✔️ Использование современных методологий написания CSS (Flexbox, семантические теги).
✔️ Чистый, читаемый код без использования тяжелых фреймворков.
